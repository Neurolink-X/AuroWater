3 world-class assets — where to place each one for maximum marketing impact.

64 × 64 px
Asset 01 / Favicon

favicon.svg

Your browser tab icon and PWA home screen icon. First visual impression on every device — keeps AuroTap instantly recognisable at tiny sizes.

Browser Tab PWA Icon Bookmark Android Shortcut
  • 🌐
    Browser TabShown in every Chrome / Safari / Firefox tab
  • 📱
    PWA / Add to Home ScreenIcon on mobile home screen when user installs site
  • 🔖
    Bookmarks & HistoryAppears next to URL in all bookmark managers
/* /public/favicon.svg */
/* In app/layout.tsx or _document.tsx */
<link rel="icon" href="/favicon.svg" type="image/svg+xml" />
<link rel="apple-touch-icon" href="/favicon.svg" />
1200 × 630 px
Asset 02 / Open Graph Hero

og-image-hero.svg

The image that appears whenever someone shares your link on WhatsApp, Instagram, Twitter/X, LinkedIn, or any messaging app. This is your single biggest free marketing surface — make it count.

WhatsApp Preview Twitter/X Card LinkedIn Share Facebook OG iMessage Link Hero Banner Email Header
  • 📤
    Social Share Preview (OG Image)Auto-shown when anyone shares aurotap.in on any platform
  • 🖼️
    Website Hero / Landing PageUse as the full-width top banner on your homepage
  • 📧
    Email Newsletter HeaderTop image in all marketing email campaigns
  • 📢
    Digital Ads (Google/Meta)Resize to 1200×628 for Facebook + Google Display ads
/* Place file at: /public/og-image-hero.svg */
/* In app/layout.tsx metadata: */
export const metadata = {
  openGraph: {
    images: [{ url: '/og-image-hero.svg',
      width: 1200, height: 630 }]
  },
  twitter: { card: 'summary_large_image',
    images: ['/og-image-hero.svg'] }
}
400 × 400 px

splash-logo.svg

Your app identity mark. Used on loading screens, app stores, and anywhere the brand needs to stand alone as a square icon. Deep navy + electric blue = premium, trustworthy, modern.

App Splash Screen Play Store Icon iOS App Icon Loading Screen WhatsApp Business DP
